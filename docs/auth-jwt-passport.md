# instalacion
```
npm i --save @nestjs/jwt passport-jwt bcrypt
npm i --save-dev @types/passport-jwt
```

# arancamos proyecto
```
npm run start:dev
```
# Modules & resource
```
nest g mo modules/auth
nest g res modules/users
```
# controllers & services
```
nest g co modules/auth
nest g s modules/auth
```
